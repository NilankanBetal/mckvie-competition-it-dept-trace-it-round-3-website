Q1)  Given an array A[ ] of positive integers of size N, where each value represents the number of chocolates in a packet. Each packet can have a variable number of chocolates. There are M students, the task is to distribute chocolate packets among M students such that :

A. Each student gets exactly one packet.

B. The difference between maximum number of chocolates given to a student and minimum number of chocolates given to a student is minimum.


Example 1:
Input: N = 8, M = 5, A = {3, 4, 1, 9, 56, 7, 9, 12}.
Output: 6.
Explanation: The minimum difference between maximum chocolates and minimum chocolates is 9 - 3 = 6 by choosing following M packets : {3, 4, 9, 7, 9}.

Example 2:
Input: N = 7, M = 3, A = {7, 3, 2, 4, 9, 12, 56}.
Output: 2.
Explanation: The minimum difference between maximum chocolates and minimum chocolates is 4 - 2 = 2 by choosing following M packets : {3, 2, 4}.

Constraints:
1 ≤ T ≤ 100
1 ≤ N ≤ 105
1 ≤ Ai ≤ 109
1 ≤ M ≤ N

Q2)  Given an array arr[] of N non-negative integers representing the height of blocks. If width of each block is 1, compute how much water can be trapped between the blocks during the rainy season.


Example 1:
Input: N = 3, arr[] = {6,9,9}.
Output: 0.
Explanation: No water will be trapped.

Example 2:
Input: N = 4, arr[] = {7,4,0,9}.
Output: 10.
Explanation: Water trapped by above block of height 4 is 3 units and above block of height 0 is 7 units. So, the total unit of water trapped is 10 units.

3 < N < 106
